QmlExplorer
===========

A Qt QML running environment and examples explorer app.



Features
===========

Controls:
- Popup
- PageView
- TabView
- ListViewEx: drop down to reflesh, drag up to load more
- ParallaxView
- SliderView
- TreeView
- Shader
- SwipeMouseArea
- ImageMouseArea
- JsonListModel
- ...

Apps:
- Clock
- Dial
- Docker
- ImageCropper
- LiveTile
- NodePad
- RoundMenu
- ....
